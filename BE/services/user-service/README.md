# User Service

Owns user-related data and behavior.

## Domain Entities

- `USER`
- `HEALTH_PROFILE`
- `DIET_PREFERENCE`
- `NUTRITION_GOAL`
- `ALLERGY`
- `FAVORITE`
- `FOOD_LOG`

## Suggested Responsibilities

- User account/profile APIs.
- Health profile management.
- Diet preference and allergy management.
- Nutrition goals.
- Favorite recipes.
- Food logs.

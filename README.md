# MSS301-Project

Monorepo managed by the root Git repository.

```text
MSS301-Project/
  FE/
    FE_app/                 Flutter application
    FE_web/                 Web frontend
  BE/
    services/
      user-service/
      recipe-service/
      ai-recommendation-service/
    shared/
```

## Applications

- `FE/FE_app`: Flutter application.
- `FE/FE_web`: web frontend.
- `BE`: backend service skeleton with user, recipe, and AI recommendation services.

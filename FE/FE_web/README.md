# NutriChef Admin Web

React/Vite admin dashboard for the food recommendation system.

## Environment

Copy `.env.example` to `.env` when the API Gateway does not use the default
`http://localhost:8080` address.

```env
VITE_API_BASE_URL=http://localhost:8080
```

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
